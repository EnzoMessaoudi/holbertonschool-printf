********                                    C - PRINTF                                       ********       

Project Description : 

The project consists of recreating the printf function.
It allows formatted text to be displayed in the terminal.
It involves handling strings, manipulating variadic arguments, 
and converting and printing different types of data.


******                                  COMPILATED COMMAND                                      ******      

                    gcc -Wall -Wextra -Werror -pedantic -std=gnu89 -Wno-format

******                                   REQUIREMENTS                                            ******

All your files will be compiled on Ubuntu 20.04 LTS using gcc, using the options -Wall -Werror -Wextra -pedantic -std=gnu89
